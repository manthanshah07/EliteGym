# EliteGym

A modern, responsive gym website built using React, Vite, Tailwind CSS, and Framer Motion. 

## Features

- **Dynamic Animations**: Smooth scroll animations and interactions built with `framer-motion`.
- **Responsive Design**: Mobile-first styling using Tailwind CSS.
- **Modern UI Components**: Utilizes `lucide-react` for beautiful iconography.
- **Fast Development**: Powered by Vite for instant server start and lightning-fast HMR.

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed on your local machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/manthanshah07/EliteGym.git
   ```

2. Navigate to the project directory:
   ```bash
   cd EliteGym
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:
```bash
npm run dev
```

This will run the app in development mode. Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
